Build It Yourself

LCE is based on the leaked Minecraft Legacy Console Edition source, compiled from the MinecraftConsoles repository. You can build it yourself using Visual Studio.

Step 1 Clone the repository
Step 2 Install Visual Studio 2022
Step 3 Open MinecraftConsoles.sln
Step 4 Build & run (Debug / Windows64)

Build Notes

Set Minecraft.Client as the Startup Project and target Windows64. Use Debug mode — Release has known render bugs. Builds for other platforms are untested and likely non-functional.
